# 行内元素有哪些？块级元素有哪些？ 空(void)元素有那些？

### 行内元素

> a、b、span、img、input、strong、select、label、em、button、textarea

### 块级元素
> div、ul、li、dl、dt、dd、p、h1-h6、blockquote

### 空元素
> 即系没有内容的HTML元素，例如：br、meta、hr、link、input、img
